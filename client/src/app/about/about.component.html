Ce este Circuitry?

Circuitry este o aplicatie de e-commerce realizata de programatori pentru programatori!
Scopul nostru este de a facilita achizitionarea celor mai noi produse din domeniul tech necesare pentru a trece cat mai repede la treaba!

Cum a inceput acest proiect?

Proiectul Circuity a inceput din dorinta de a crea un proiect simplu de atestat pentru informatica, extinzandu-se ulterior din dorinta developer-ului de a invata mai multe lucruri si a le aplica pentru a crea un website potrivit cerintelor sale.

Ce tehnologii au fost folosite pentru a crea Circuitry?

Pentru Circuitry, tehnologiile folosite au fost urmatoarele:

  • Frontend: Angular, Bootstrap 5, Font Awesome 5
  • Backend: .NET - Limbaj de Programare: C#
  • Databases: SQLite + Redis
  • Procesare de plati: Stripe
